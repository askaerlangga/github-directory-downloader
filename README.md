# GitHub Directory Downloader (Web Statis)

✨ **Cobain Langsung:** [https://www.askaerlangga.my.id/github-directory-downloader](https://www.askaerlangga.my.id/github-directory-downloader)

Aplikasi ini memudahkan kamu buat unduh folder tertentu dari repositori GitHub langsung jadi file ZIP, nggak perlu ribet klon satu repo penuh!

## Cara Pakai
1. Buka [link ini](https://www.askaerlangga.my.id/github-directory-downloader) di browser kamu.
2. Tempel URL folder GitHub yang mau kamu unduh.
   - Contoh: `https://github.com/OCA/web/tree/18.0/web_responsive`
3. Klik tombol **Unduh ZIP**.
4. Tunggu sebentar sampai prosesnya selesai, file ZIP bakal otomatis masuk ke folder unduhan kamu.

## Fitur Unggulan
- **Full Frontend:** Nggak pakai server backend, semuanya diproses langsung di browser kamu.
- **Sat-set:** Pakai GitHub Git Trees API biar ambil datanya lebih ngebut dan efisien.
- **Modern UI:** Desain responsif dengan gaya *Glassmorphism* yang sejuk di mata.
- **Kompresi Lokal:** Proses ZIP dilakukan di sisi klien pakai library JSZip.

## Catatan Penting
- Batas GitHub API cuma **60 request/jam** (karena nggak butuh login atau token).
- Cuma berfungsi buat **Repositori Publik** ya.
