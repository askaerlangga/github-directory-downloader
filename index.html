Cepat & Lokal

Proses ZIP dilakukan langsung di browser kamu. Data nggak bakal dikirim ke server mana pun.

Privasi Aman

Cuma pakai GitHub API buat ambil konten publik secara anonim tanpa perlu login atau token.

Kenapa pakai ini?

  • Nggak Perlu Instal: Langsung jalan di browser, nggak butuh Git atau aplikasi lain.
  • Pilih Folder Aja: Unduh cuma bagian yang kamu butuhin, nggak perlu tarik satu repo.
  • Proses Ngebut: Pakai teknologi Git Trees biar ambil datanya sat-set.
  • ZIP Instan: File ZIP dibuat otomatis di perangkat kamu pakai JSZip.

Hal yang perlu diingat

  • Cuma bisa buat Repositori Publik ya.
  • Batas API GitHub: 60 request/jam (karena tanpa login).
  • Kecepatan unduh tergantung internet kamu dan jumlah filenya.
  • URL-nya harus lengkap, ada bagian /tree/BRANCH/... nya.
