Cepat & Lokal

Proses ZIP dilakukan langsung di browser Anda. Tidak ada data yang dikirim ke server eksternal.

Privasi Terjaga

Kami hanya menggunakan GitHub API untuk mengambil konten publik secara anonim.

Fitur Utama

  • Tanpa Instalasi: Berjalan langsung di browser tanpa perlu Git.
  • Selektif: Unduh hanya folder yang Anda butuhkan, bukan seluruh repo.
  • Ringan: Menggunakan struktur Git Trees untuk pengambilan data cepat.
  • Kompresi Lokal: Pembuatan ZIP dilakukan di sisi klien (JSZip).

Catatan Penting

  • Hanya mendukung Repositori Publik.
  • Limit API GitHub: 60 request/jam (Unauthenticated).
  • Kecepatan tergantung pada jumlah file dan koneksi internet Anda.
  • Format URL harus menyertakan bagian /tree/BRANCH/...
